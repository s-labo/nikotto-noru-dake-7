# にこっと乗るだけ7｜デジタルパンフレット

株式会社カーサービス山形が提供する新車4WD軽自動車の月々定額カーリース
**「にこっと乗るだけ7」** のサービス紹介用デジタルパンフレット（静的サイト）です。

来店した新規のお客様、およびオンラインでサービス内容を理解いただくことを目的としています。
**本のようにページをめくる**デジタルパンフレット（デスクトップは見開き2ページ／スマホは単ページ）で、ボタン・矢印キー・スワイプ・画面タップでめくれます。

## ページ構成（全17見開き相当）

1. 表紙
2. カーサービス山形の強み
3. にこっと乗るだけ7とは（コミコミ説明）
4. お支払い方法と料金プラン早見
5〜7. 料金プラン PLAN1〜3（各6車種・全18車種の写真付き）
8. お得なパックプラン（メンテ／セーフティー／Wパック）
9. カーリースのしくみ・満了時の4つの選択肢
10. 他社比較表
11. お客様の声
12〜13. よくあるご質問（FAQ）
14. CSY 中扉
15〜16. 店舗のご案内（山形県内9店舗・各店舗サイトへのリンク付き）
17. 会社概要
18. 理念と沿革
19. 社長メッセージ

## 構成

```
index.html        本体（HTML+CSS+JS すべて内包・ページめくりエンジン）
assets/cars/      車種画像（teigaku1〜3 × car1〜6 / 全18枚）
assets/president.jpg  代表写真
```
