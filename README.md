# にこっと乗るだけ7｜デジタルパンフレット

株式会社カーサービス山形が提供する新車4WD軽自動車の月々定額カーリース
**「にこっと乗るだけ7」** のサービス紹介用デジタルパンフレット（静的サイト）です。

来店した新規のお客様、およびオンラインでサービス内容を理解いただくことを目的としています。

## 内容

- ヒーロー（コミコミ訴求・月々11,000円〜）
- 料金プラン3種（各6車種・全18車種の写真付き）
- お得なパックプラン（メンテナンスパック／セーフティーパック／Wパック）
- 月々定額カーリースの仕組み・リース満了時の4つの選択肢
- 他社比較表
- お客様の声
- よくある質問（FAQ）
- 店舗情報（山形県内9店舗）

## 構成

```
index.html        本体（1ページ）
styles.css        スタイル
assets/cars/      車種画像（teigaku1〜3 × car1〜6）
```
